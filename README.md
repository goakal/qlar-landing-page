# Qlar Landing Page

Static landing page for Qlar — AI Agent for Indonesian Clinics.

Hosted via GitHub Pages: see the **Pages** section of the repo settings, or visit the deployed URL listed under the repo's "About" panel.

## Local preview

Open `index.html` directly in a browser, or serve the directory with any static server:

```bash
python -m http.server 8000
# then open http://localhost:8000
```
